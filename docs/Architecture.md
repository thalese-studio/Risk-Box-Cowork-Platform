# 1. 系統定位

Risk Box 為部署於 **OSI L2 / L3 協議層** 的金融風險節點。

特點：

- 不修改核心銀行系統
- 不侵入帳務流程
- 支援 ISO-8583 / ISO-20022
- 可隱身部署（NO-IP）
- 即時風險標記
- 可選擇性主動干預

# 5. 風險演算模型

## 5.1 核心公式

R_score 計算公式：

R_score = Σ(W_i(t) × δ_i) + Γ

其中：

W_i(t) ＝ 時間動態權重  
δ_i ＝ 特徵偏離值  
Γ ＝ 聯防共振因子

# 7. 信心度與聯防共振

## 7.1 信心度公式

C = Σ(T_j × S_j) / √(N_rep + ε)

T_j = 節點信任度  
S_j = 行為相似度  
N_rep = 回報數量  

## 7.2 聯防共振因子

Γ = f(跨節點同步數, 信任度, 相似度)

若：

- 3家以上銀行標記
- 相似度 > 0.8

則 Γ ≥ +20
