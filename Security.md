# Security Policy

## 責任研究與揭露 (Responsible Disclosure)
Risk Box 是一個旨在提升金融體系透明度與安全性的開源架構研究。我們極度重視系統的安全性與健壯性。

## 支持版本
我們僅對當前 GitHub 分支上的最新架構模型提供技術討論支援。

## 漏洞回報
如果您在 Risk Box 的演算邏輯、ISO-8583 權值模型或物理層繞過防護中發現任何安全缺陷：
1. 請勿公開發布 PoC (概念證明) 攻擊。
2. 請透過加密郵件聯繫核心開發組：`thalese.studio@gmail.com` (或透過 GitHub Issue 提交匿名技術質疑)。
3. 我們將評估該漏洞對「聯防網路」的影響，並在必要時更新 $C$ (信心度指數) 演算邏輯。

## 免責聲明
本研究不直接營運任何實體金融服務。任何因第三方錯誤實作本架構而導致的損失，本研究組概不負責。
